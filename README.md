# Actividad entregable 1 - Documentación y refactorización

## Descripción
Programa en Java que analiza un array de enteros:
- Indica si el valor máximo se repite más de una vez ("SI"/"NO").
- Calcula la media del array.
- Evalúa la media: "BIEN" si media >= 5, en caso contrario "MAL".

## Requisitos
- Java (JDK) 8 o superior.

## Ejecución
Compilar:

```bash
javac -d . src/ejerciciosexamen/AnalisisArray.java 
```

Ejecutar
```bash
java ejerciciosexamen.AnalisisArray
```
